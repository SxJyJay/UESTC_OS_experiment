# UESTC_OS_experiment
电子科技大学操作系统进程与资源管理实验代码（python）  
功能实现：实现了实验要求中的全部功能  
  指令命令与功能对应关系如下：  
	cr PID priority  ----  创建新进程   
	de PID  ----  删除进程及其子进程  
	to  ----  超时（模拟时间片轮转）  
	req RID num  ----  （正在运行的进程）请求资源  
	rel PID  ----  （正在运行的进程）释放资源  
	lsp  ----  列出当前系统中所有进程的信息  
	lsr	 ----  列出当前系统中所有资源的信息  
  cp PID  ----  根据PID查询进程信息
